Section 1 & 2 — Overview & Problem Verdant Documentation
Parichay

Verdant Kya Hai?

Verdant ek web-based platform hai jo Indian college students ke daily carbon footprint ko track, visualize aur reduce karne mein help karta hai — gamification ke through.

🌿 Core Mission (Mukhya Uddeshya)

Har student ka daily carbon footprint measure karna — travel, food, energy aur waste ke basis par — aur unhein eco-friendly choices lene ke liye motivate karna through points, leaderboard aur AI recommendations.

Samasya

Problem Statement

📌 Official Problem Statement

"Design a solution that helps individuals understand, track, and reduce their carbon footprint through simple actions and personalized insights."

✅ Verdant Is Problem Ko Kaise Solve Karta Hai?

Problem Statement Ka Hissa Verdant Feature Screen
🧠 "Understand" — samajhna Category Breakdown chart, You vs Campus comparison bar, Timeline sparkline Insights
📝 "Track" — track karna Daily Score Gauge (0–100), Weekly bar chart Mon–Sun, Activity Feed with timestamps, CO₂ savings counter Dashboard
📉 "Reduce" — kam karna AI Green Targets: Cycling −40kg, Plant-Based Days −15kg, Phantom Load −5kg. Campus Map eco check-ins. Insights + Map
⚡ "Simple actions" — aasaan kaam 1-click Quick Log buttons (Bottle +5pts, Bus +15pts, Vegan +10pts). Custom log bhi possible. Quick Log
🎯 "Personalized insights" — personal Baseline Quiz — commute, diet, energy answers se har user ka alag footprint aur AI suggestions Baseline Quiz

🎤 Presentation Mein Aise Bolein

"Our problem statement was — 'Design a solution that helps individuals understand, track, and reduce their carbon footprint through simple actions and personalized insights.' Verdant addresses every part: the Insights page helps users understand their footprint visually through charts, the Dashboard tracks it in real-time with daily goals, AI Targets actively guide reduction with quantified savings, Quick Log makes eco-actions possible in one click, and the Baseline Quiz makes everything personalized to each individual user."

💾 Data Persistence (Data Kahan Save Hota Hai?)

Saare points, level, profile (naam, avatar, bio), activity logs — sab kuch localStorage mein save hota hai. Matlab page refresh karne ke baad bhi sab data rehta hai. Koi backend server nahi chahiye — purely frontend-based storage.

📊 Daily Score System

Dashboard par ek circular gauge dikhta hai (0 se 100). Har activity se score badhta hai. 100 reach karne par "Daily Target Met! Stellar Stewardship!" message dikhta hai. Isse user ko rozana app kholine ka reason milta hai.

Section 6 — Profile System & Deployment Verdant Documentation
Profile System

Profile Edit Feature

User apna naam, avatar aur bio change kar sakta hai — sab kuch localStorage mein save hota hai.

🖼️ Avatar Options

6 preset Unsplash photos mein se choose kar sakte hain, ya apni custom image URL paste kar sakte hain. Avatar sahi na load ho toh fallback image automatically aata hai.

✏️ Kahan se Open Karein?

Desktop: Sidebar par profile card pe hover karo → pencil icon aayega → click karo. Top-bar: Apna avatar (top-right) pe click karo. Dono se modal khulta hai.

Deploy Karna

Netlify pe Deploy Guide

Verdant ko publicly live karna bahut aasaan hai — sirf 5 minutes lagte hain.

1

Dependencies Install Karein

Terminal mein project folder mein jaao aur npm install run karo. Isse saare required packages install ho jaayenge.

2

Production Build Banao

npm run build command se dist/ folder banega. Isme optimized HTML, CSS aur JS files hongi — 303kb JS + 61kb CSS total size.

3

Netlify pe Upload Karo

app.netlify.com pe jaao → Login karo → Dashboard pe dist folder drag & drop karo. Ya GitHub se connect karke auto-deploy setup karo.

4

GitHub se Connect Karo

Netlify Dashboard → "Add new site" → "Import an existing project" → GitHub se Dev-Yash502/Verdant repo select karo. Build command: npm run build, Publish dir: dist. Auto-deploy ho jaayega har commit pe.

5

App Live!

Netlify ek random URL dega jaise verdant-xyz.netlify.app. Ise share karo. Custom domain bhi add kar sakte ho. GitHub repo: github.com/Dev-Yash502/Verdant

⚙️ Build Settings (GitHub Connect ke liye)

Build Command: npm run build     Publish Directory: dist     Node Version: 18+

Verdant – Project Summary

Ek pankti mein: Verdant woh platform hai jo Indian college students ko unke carbon footprint ke baare mein aware karta hai aur eco-friendly choices ke liye reward deta hai.

✅ Kya Achieve Kiya?

  • 7 fully functional screens with smooth navigation
  • Personalized carbon footprint based on user quiz answers
  • Real-time points & level system with localStorage persistence
  • Logic-driven smart reduction recommendations (Insights page)
  • Campus & Student leaderboard system
  • Interactive Campus Map with eco-location check-ins
  • 5-question Eco Quiz with explanations
  • Profile edit system (naam, avatar, bio)
  • Fully responsive — desktop aur mobile dono ke liye
  • Production build ready (Netlify deploy tested)

🎯 Unique Features

  • India-specific carbon emission factors use kiye gaye
  • IIT/NIT/BITS campus context — relevant to Indian students
  • Gamification: points, levels, daily streak goals
  • Claymorphism design system — premium feel
  • Quick Log: 3-click activity logging
  • Baseline quiz se personalized starting footprint
  • Weekly bar chart + sparkline analytics
  • Campus vs. You comparison bar
  • No backend needed — fully frontend
  • No backend/API key needed — fully frontend
